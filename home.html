Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How Do you Select an element from the DOM ?

Using getElementById: The getElementById method allows to select an element by its ID

What is event delegation in the context of the DOM and why it is useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated

How do you manipulate an element attributes an styles using the DOM?

The classList object only provide methods to change the class of an element. If we want to change other attributes like id , href , or src , we can use the setAttribute() method.
